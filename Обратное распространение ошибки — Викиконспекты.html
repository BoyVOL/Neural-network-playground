Обратное распространение ошибки

Обучение как задача оптимизации[править]

Рассмотрим простую нейронную сеть без скрытых слоев, с двумя входными вершинами и одной выходной, в которых каждый нейрон использует линейную функцию активации, (обычно, многослойные нейронные сети используют нелинейные функции активации, линейные функции используются для упрощения понимания) которая является взвешенной суммой входных данных.
Простая нейронная сеть с двумя входными вершинами и одной выходной

Изначально веса задаются случайно. Затем, нейрон обучается с помощью тренировочного множества, которое в этом случае состоит из множества троек (x1,x2,t)(x1,x2,t) где x1x1 и x2x2 — это входные данные сети и tt — правильный ответ. Начальная сеть, приняв на вход x1x1 и x2x2, вычислит ответ yy, который вероятно отличается от tt. Общепринятый метод вычисления несоответствия между ожидаемым tt и получившимся yy ответом — квадратичная функция потерь:

E=(t−y)2,E=(t−y)2, где EE ошибка.
В качестве примера, обучим сеть на объекте (1,1,0)(1,1,0), таким образом, значения x1x1 и x2x2 равны 1, а tt равно 0. Построим график зависимости ошибки EE от действительного ответа yy, его результатом будет парабола. Минимум параболы соответствует ответу yy, минимизирующему EE. Если тренировочный объект один, минимум касается горизонтальной оси, следовательно ошибка будет нулевая и сеть может выдать ответ yy равный ожидаемому ответу tt. Следовательно, задача преобразования входных значений в выходные может быть сведена к задаче оптимизации, заключающейся в поиске функции, которая даст минимальную ошибку.
График ошибки для нейрона с линейной функцией активации и одним тренировочным объектом

В таком случае, выходное значение нейрона — взвешенная сумма всех его входных значений:

y=x1w1+x2w2,y=x1w1+x2w2,

где w1w1 и w2w2 — веса на ребрах, соединяющих входные вершины с выходной. Следовательно, ошибка зависит от весов ребер, входящих в нейрон. И именно это нужно менять в процессе обучения. Распространенный алгоритм для поиска набора весов, минимизирующего ошибку — градиентный спуск. Метод обратного распространения ошибки используется для вычисления самого "крутого" направления для спуска.

Дифференцирование для однослойной сети[править]

Метод градиентного спуска включает в себя вычисление дифференциала квадратичной функции ошибки относительно весов сети. Обычно это делается с помощью метода обратного распространения ошибки. Предположим, что выходной нейрон один, (их может быть несколько, тогда ошибка — это квадратичная норма вектора разницы) тогда квадратичная функция ошибки:

E=12(t−y)2,E=12(t−y)2, где EE — квадратичная ошибка, tt — требуемый ответ для обучающего образца, yy — действительный ответ сети.

Множитель 1212 добавлен чтобы предотвратить возникновение экспоненты во время дифференцирования. На результат это не повлияет, потому что позже выражение будет умножено на произвольную величину скорости обучения (англ. learning rate).

Для каждого нейрона jj, его выходное значение ojoj определено как

oj=φ(netj)=φ(∑nk=1wkjok).oj=φ(netj)=φ(∑k=1nwkjok).

Входные значения netjnetj нейрона — это взвешенная сумма выходных значений okok предыдущих нейронов. Если нейрон в первом слое после входного, то okok входного слоя — это просто входные значения xkxk сети. Количество входных значений нейрона nn. Переменная wkjwkj обозначает вес на ребре между нейроном kk предыдущего слоя и нейроном jj текущего слоя.

Функция активации φφ нелинейна и дифференцируема. Одна из распространенных функций активации — сигмоида:

φ(z)=11+e−zφ(z)=11+e−z

у нее удобная производная:

dφ(z)dz=φ(z)(1−φ(z))dφ(z)dz=φ(z)(1−φ(z))

Находим производную ошибки[править]

Вычисление частной производной ошибки по весам wijwij выполняется с помощью цепного правила:

∂E∂wij=∂E∂oj∂oj∂netj∂netj∂wij∂E∂wij=∂E∂oj∂oj∂netj∂netj∂wij

Только одно слагаемое в netjnetj зависит от wijwij, так что

∂netj∂wij=∂∂wij(∑nk=1wkjok)=∂∂wijwijoi=oi.∂netj∂wij=∂∂wij(∑k=1nwkjok)=∂∂wijwijoi=oi.

Если нейрон в первом слое после входного, то oioi — это просто xixi.

Производная выходного значения нейрона jj по его входному значению — это просто частная производная функции активации (предполагается что в качестве функции активации используется сигмоида):

∂oj∂netj=∂∂netjφ(netj)=φ(netj)(1−φ(netj))∂oj∂netj=∂∂netjφ(netj)=φ(netj)(1−φ(netj))

По этой причине данный метод требует дифференцируемой функции активации. (Тем не менее, функция ReLU стала достаточно популярной в последнее время, хоть и не дифференцируема в 0)

Первый множитель легко вычислим, если нейрон находится в выходном слое, ведь в таком случае oj=yoj=y и

∂E∂oj=∂E∂y=∂∂y12(t−y)2=y−t∂E∂oj=∂E∂y=∂∂y12(t−y)2=y−t

Тем не менее, если jj произвольный внутренний слой сети, нахождение производной EE по ojoj менее очевидно.

Если рассмотреть EE как функцию, берущую на вход все нейроны L=u,v,…,wL=u,v,…,w получающие на вход значение нейрона jj,

∂E(oj)∂oj=∂E(netu,netv,…,netw)∂oj∂E(oj)∂oj=∂E(netu,netv,…,netw)∂oj

и взять полную производную по ojoj, то получим рекурсивное выражение для производной:

∂E∂oj=∑ℓ∈L(∂E∂netℓ∂netℓ∂oj)=∑ℓ∈L(∂E∂oℓ∂oℓ∂netℓwjℓ)∂E∂oj=∑ℓ∈L(∂E∂netℓ∂netℓ∂oj)=∑ℓ∈L(∂E∂oℓ∂oℓ∂netℓwjℓ)

Следовательно, производная по ojoj может быть вычислена если все производные по выходным значениям oℓoℓ следующего слоя известны.

Если собрать все месте:

∂E∂wij=δjoi∂E∂wij=δjoi

и

δj=∂E∂oj∂oj∂netj={(oj−tj)oj(1−oj)(∑ℓ∈Lwjℓδℓ)oj(1−oj)если j является нейроном выходного слоя,если j является нейроном внутреннего слоя.δj=∂E∂oj∂oj∂netj={(oj−tj)oj(1−oj)если j является нейроном выходного слоя,(∑ℓ∈Lwjℓδℓ)oj(1−oj)если j является нейроном внутреннего слоя.

Чтобы обновить вес wijwij используя градиентный спуск, нужно выбрать скорость обучения, η>0η>0. Изменение в весах должно отражать влияние EE на увеличение или уменьшение в wijwij. Если ∂E∂wij>0∂E∂wij>0, увеличение wijwij увеличивает EE; наоборот, если ∂E∂wij<0∂E∂wij<0, увеличение wijwij уменьшает EE. Новый ΔwijΔwij добавлен к старым весам, и произведение скорости обучения на градиент, умноженный на −1−1, гарантирует, что wijwij изменения будут всегда уменьшать EE. Другими словами, в следующем уравнении, −η∂E∂wij−η∂E∂wij всегда изменяет wijwij в такую сторону, что EE уменьшается:

Δwij=−η∂E∂wij=−ηδjoiΔwij=−η∂E∂wij=−ηδjoi

Алгоритм[править]

  • ηη - скорость обучения
  • αα - коэффициент инерциальности для сглаживания резких скачков при перемещении по поверхности целевой функции
  • {xdi,td}n,mi=1,d=1{xid,td}i=1,d=1n,m — обучающее множество
  • stepssteps — количество повторений
  • network(x)network(x) — функция, подающая x на вход сети и возвращающая выходные значения всех ее узлов
  • layerslayers — количество слоев в сети
  • layerilayeri — множество нейронов в слое i
  • outputoutput — множество нейронов в выходном слое


fun BackPropagation(η,α,{xdi,td}n,mi=1,d=1,steps)(η,α,{xid,td}i=1,d=1n,m,steps):
   init {wij}i,j{wij}i,j
   repeat stepssteps:
       for dd = 11 to mm:
           oo = network({xdi})network({xid}) 
           for k∈outputk∈output:
               δkδk = σ′(ok)(tk−ok)σ′(ok)(tk−ok)
           for ii = layers−1layers−1 to 11:
               for j∈layerij∈layeri:
                   δjδj = σ′(oj)∑k∈layeri+1δkwj,kσ′(oj)∑k∈layeri+1δkwj,k
           for ∀wi,j∀wi,j:
               Δwni,jΔwi,jn = αΔwn−1i,j+(1−α)ηδjoiαΔwi,jn−1+(1−α)ηδjoi
               wni,jwi,jn = wn−1i,j+Δwni,jwi,jn−1+Δwi,jn
   return wijwij

Недостатки алгоритма[править]

Несмотря на многочисленные успешные применения обратного распространения, оно не является универсальным решением. Больше всего неприятностей приносит неопределённо долгий процесс обучения. В сложных задачах для обучения сети могут потребоваться дни или даже недели, она может и вообще не обучиться. Причиной может быть одна из описанных ниже.

Градиентный спуск может найти локальный минимум вместо глобального

Паралич сети[править]

В процессе обучения сети значения весов могут в результате коррекции стать очень большими величинами. Это может привести к тому, что все или большинство нейронов будут функционировать при очень больших выходных значениях, а производная активирующей функции будет очень мала. Так как посылаемая обратно в процессе обучения ошибка пропорциональна этой производной, то процесс обучения может практически замереть.

Локальные минимумы[править]

Градиентный спуск с обратным распространением ошибок гарантирует нахождение только локального минимума функции; также, возникают проблемы с пересечением плато на поверхности функции ошибки.
